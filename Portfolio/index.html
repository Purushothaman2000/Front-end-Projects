Project Title 2

I developed a basic calculator using HTML, CSS, and JavaScript. The calculator performs standard arithmetic operations like addition, subtraction, multiplication, and division. It features a clean and responsive interface with a modern design, ensuring usability across different devices. Project Title 3

I built a responsive weather app using HTML, CSS, and JavaScript. The app fetches real-time weather data from a public API, providing users with current temperature, humidity, and weather conditions for any searched location. It features a clean and user-friendly interface, with visual cues like weather icons and background changes to match the weather conditions. The app is designed to work seamlessly on various devices, ensuring a smooth user experience.
